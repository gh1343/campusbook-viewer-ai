      2. 2. 목적중심의 행동변화를 설계한다
      3. 3. 진단기반의 구조화된 대화를 한다
      4. 4. 코칭 사례: 역할에 맞게 리더십 행동 바꾸기
      5. 생각 파트너의 심리코칭
    2. 제2장 변화 요구를 변화로 끌어낸다
      1. 1. 사람들은 강요된 변화에 저항한다
      2. 2. 긍정적 이탈로 새로운 변화를 끌어낸다
      3. 3. 자기성찰로 행동변화를 점검한다
      4. 4. 코칭 사례: 팀 변화로 개인 변화 이끌기
      5. 생각 파트너의 심리코칭

      2. 2. 실행의도가 약하면 결과를 놓친다
      3. 3. 결과를 만드는 상호협력을 끌어낸다
      4. 4. 코칭 사례: 팀장의 성과리더십 향상
      5. 생각 파트너의 심리코칭
    4. 제4장 변화에 통합적으로 접근한다
      1. 1. 코칭 리더십 프레임워크로 변화를 만든다
      2. 2. 존재보다 실행을 먼저 다룬다
      3. 3. 생산자보다 조력자 역할을 한다
      4. 4. 긍정성으로 생산성을 촉진한다
      5. 5. 코칭 사례: 통합적 코칭의 도입과 운영
      6. 생각 파트너의 심리코칭
  1. 제2부 생각을 바꾸면 행동과 결과가 바뀐다
    1. 제5장 자기인식을 강화한다
      1. 1. 자기인식을 깨워 변화를 촉진한다
      2. 2. 가정적 사고에서 벗어난다
      3. 3. 개인의 자기중심성을 극복한다
      4. 4. 부정적 감정에 쉽게 휘둘리지 않는다
      5. 5. 코칭 사례: 이혼 위기에 직면한 부부
      6. 생각 파트너의 심리코칭
    2. 제6장 관점확대로 효과성을 높인다
      1. 1. 관점확대로 사고의 유연성을 키운다
      2. 2. 입장 바꿔 보기로 관점 차이를 해소한다
      3. 3. 삶의 역경을 다른 관점에서 본다
      4. 4. 코칭 사례: 소중한 것의 재발견
      5. 생각 파트너의 심리코칭
    3. 제7장 자기확신을 고양한다
      1. 1. 강점발견으로 자기확신을 높인다
      2. 2. 자기확신에 묶이지 않는다
      3. 3. 역할보다 성향 의존을 경계한다
      4. 4. 코칭 사례: 숨은 강점의 발견
      5. 생각 파트너의 심리코칭

      3. 3. 통찰 심화로 더 나은 해법을 찾는다
      4. 4. 코칭 사례: 통합적 자기 체험
      5. 생각 파트너의 심리코칭
